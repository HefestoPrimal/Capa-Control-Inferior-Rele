# Activador Inferior para Relé

Esta PCB esta diseñada para soldarse directamente a la parte inferior de un relevador y manejarlo directamente con un microcontrolador a través de 2 pines de alimentación y uno de control, sumándole también 3 pines de salida correspondiente al COMUN, NORMALMENTE ABIERTO y NORMALMENTE CERRADO.
Su principal ventaja es el espacio utilizado y el hecho de contar con todos los componentes para una fase de potencia que proteja al microcontrolador de corrientes parásitas generadas por el mismo relevador.
